青岛市2022编程市赛 T3结论与简易证明

2022-08-08 oi

首先，结论与证明来自http://oeis.org/A005732/a005732.pdf，这里给出简单翻译和加工

给出结论，在一个圆上取 nnn 个点相连，构成的三角形个数为 Cn+36+Cn+15+Cn5C_{n+3}^6+C_{n+1}^5+C_n^5Cn+36​+Cn+15​+Cn5​

如何证明呢，我们不妨从圆上n个点中的任意个可以构成几个三角形来入手，我们先看7个点的图，来观察一下!

如果你有十足的耐心的话，你可以数出来，这是 287287287 个三角形

但是，我们要求出一个通项公式，我们可以发现，每一个三角形都是由多个或 000 个圆上的点组成的，让我们通过这个分类讨论

首先tips：认识到圆“上"的概念：就是在圆的边上

  1. 由圆上 333 个点构成的三角形有 Cn3C_n^3Cn3​ 个。证明：圆上每三个点可以构成一个三角形，反之亦然.（这个不用画图了吧qwq）
  2. 只由圆上 222 个点 和另外 111 个点 构成的三角形个数有 4×Cn44 \times C_n^44×Cn4​ 个。证明：对于圆上的每 444 个点，可以构成 444 个"有 222 个点在圆上" 的三角形
  3. 只由圆上 111 个点 和另外 222 个点 构成的三角形个数有 5×Cn55 \times C_n^55×Cn5​个。证明：对于圆上的每 555 点，可以构成 555 个"有 111 个点在圆上" 的三角形
  4. 只由圆上 000 个点 和另外 333 个点 构成的三角形个数有 Cn6C_n^6Cn6​个。证明：对于圆上的每 666 点，可以构成 111 个"有 000 个点在圆上" 的三角形 （当然，这个图里还有别点之间的的连线，只不过我懒得画了emmmm）

最后，就是整理这个式子了（挺复杂的qwq）（参考this）
∵Cn+1m=Cnm+Cnm−1\because C_{n+1}^m = C_{n}^m + C_{n}^{m-1}∵Cn+1m​=Cnm​+Cnm−1​

∴Cn3+4×Cn4+5×Cn5+Cn6\therefore C_n^3 + 4 \times C_n^4 + 5 \times C_n^5 + C_n^6∴Cn3​+4×Cn4​+5×Cn5​+Cn6​

=Cn3+4×Cn4+4×Cn5+Cn5+Cn6C_n^3 + 4 \times C_n^4 + 4 \times C_n^5 + C_n^5 + C_n^6Cn3​+4×Cn4​+4×Cn5​+Cn5​+Cn6​

=Cn3+4×Cn+15+Cn+16C_n^3 + 4 \times C_{n+1}^5 + C_{n+1}^6Cn3​+4×Cn+15​+Cn+16​

=Cn3+3×Cn+15+Cn+15+Cn+16C_n^3 + 3 \times C_{n+1}^5 + C_{n+1}^5 + C_{n+1}^6Cn3​+3×Cn+15​+Cn+15​+Cn+16​

=Cn3+3×Cn+15+Cn+26C_n^3 + 3 \times C_{n+1}^5 + C_{n+2}^6Cn3​+3×Cn+15​+Cn+26​

=Cn3+Cn+15+2×Cn+15+Cn+26C_n^3 + C_{n+1}^5 + 2 \times C_{n+1}^5 + C_{n+2}^6Cn3​+Cn+15​+2×Cn+15​+Cn+26​

=Cn3+Cn4+Cn5+2×Cn+15+Cn+26C_n^3 + C_n^4 + C_n^5 + 2 \times C_{n+1}^5 + C_{n+2}^6Cn3​+Cn4​+Cn5​+2×Cn+15​+Cn+26​

=Cn+14+Cn5+2×Cn+15+Cn+26C_{n+1}^4 + C_n^5 + 2 \times C_{n+1}^5 + C_{n+2}^6Cn+14​+Cn5​+2×Cn+15​+Cn+26​

=Cn+25+Cn+26+Cn5+Cn+15C_{n+2}^5 + C_{n+2}^6+C_n^5+C_{n+1}^5Cn+25​+Cn+26​+Cn5​+Cn+15​

=Cn+36+Cn5+Cn+15C_{n+3}^6 + C_n^5+C_{n+1}^5Cn+36​+Cn5​+Cn+15​
